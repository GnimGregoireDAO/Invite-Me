Comment installer l'application ?

1

Téléchargement

Cliquez sur le bouton "Télécharger pour Android" ci-dessus pour obtenir le fichier APK de l'application.

2

Autoriser les sources inconnues

Dans les paramètres de votre téléphone, autorisez l'installation d'applications provenant de sources inconnues.

3

Installation

Ouvrez le fichier téléchargé et suivez les instructions pour installer Inv@teMe sur votre appareil.

4

Première utilisation

Lancez l'application et créez votre compte pour commencer à organiser vos événements !

Questions Fréquentes

Oui, Inv@teMe est entièrement gratuite. Aucun frais caché, aucun abonnement requis.
Une connexion Internet est requise pour synchroniser vos événements et interagir avec d'autres utilisateurs.
